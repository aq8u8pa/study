## 전체 학습 계획 (예정)

* 사용 자료: [**점프 투 파이썬**](https://wikidocs.net/book/1)

* 1월 23일 이전 웹 파트 전환은 없음

### 2월 : Web
* **Flask + HTML** 또는 **Django + React** 중 선택
* 초보자 비중을 고려해 Flask + HTML 우선 고려

* 코드 리뷰 제공 예정

### 3월 : 팀 프로젝트 (희망자 있는 경우)

* 학습용 프로젝트 (상업성/완성도/심미성은 중요하지 않음)
* 일정 및 범위는 참여자 의견에 따라 조정
